- **Дословный бриф (источник истины):** `docs/PRODUCT/SOURCE_BRIEF.md` — не редактировать.
  Производные: `docs/PRODUCT/REQUIREMENTS.md` (FR/NFR/EP), `docs/ARCHITECTURE/INVARIANTS.md`.
- **Git remote:** `git@github.com:DavidGarguliya/amobile-test.git` (ветка `main`).
- **Замечание:** локальная папка называется `amobail-test` (через «a»), удалённый репозиторий —
  `amobile-test` (через «i»). Использовать remote как задан.
